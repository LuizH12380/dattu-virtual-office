# Escritório Virtual Dattu — Índice Central

Este vault é o **cérebro da empresa**. Cada nota é um neurônio. Os links entre elas formam a rede neural de conhecimento.

## Seções

- [[Time/]] — Colaboradores, contratações, desligamentos, perfis
- [[Decisoes/]] — Decisões estratégicas tomadas pelos agentes
- [[Ideias/]] — Brainstorms, oportunidades, inovações
- [[Reunioes/]] — Atas e resumos de reuniões
- [[Criativos/]] — Briefs, conceitos e outputs criativos
- [[Processos/]] — Documentação de processos e workflows

| Agente | Função | Responsabilidades |

## Como funciona

1. O **Orquestrador** recebe uma tarefa
2. Identifica qual(is) agente(s) são responsáveis
3. Os agentes lêem o contexto relevante do vault
4. Executam a tarefa usando Claude como motor
5. Gravam os resultados de volta no vault como novas notas
6. Criam links entre notas relacionadas — formando a rede neural
